# Intelligent Agents (ST6058CEM)

This repository contains materials and coursework for the Intelligent Agents (ST6058CEM) module. The primary aim is to explore the fundamental concepts of Artificial Intelligence and knowledge-based systems. Through this module, we analyze the complexity of typical AI problems, critically evaluate various methods for searching, planning, and knowledge representation, and assess their ethical applications to real-world scenarios.

At the core of the study is the design of Intelligent Agents using the PEAS framework (Performance, Environment, Actuators, and Sensors). Key topics covered include state spaces, heuristics, constraints, and the distinction between deductive and abductive reasoning. By building agents that perceive their environment and act rationally to achieve goals, we learn how to map complex problems into solvable logical and computational models.

The practical implementations and concepts heavily reference the "Artificial Intelligence: A Modern Approach" (AIMA) textbook and its associated open-source `aima-python` repository. These resources help bridge the gap between theoretical knowledge and functional programming, providing a robust foundation in modern AI vocabulary and methodologies.
